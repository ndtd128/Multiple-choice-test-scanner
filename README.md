# Multiple Choice Test Auto Grader
## Usage
1. Print answer sheet: [Sample](/sampleAnswerSheet/sample.jpg)
2. Install prerequisite libraries
```sh
pip install -r .\requirements.txt
```

3. Change to project directory and run
```sh
python main.py
```

## Technology
Python and openCV
